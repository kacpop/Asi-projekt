# Asi-projekt

Projekt miał na celu stworzenie aplikacji webowej, która pozwalałaby na przechodzenie między różnymi stronami z różnymi funkcjami. Wejście do aplikacji wymaga rejestracji, póżniej logowania użytkowanika.
